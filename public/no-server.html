Pomodoro App - Setup Required

The Pomodoro Timer application requires a local server to function properly. It seems you're trying to open the HTML files directly, which won't work because of browser security restrictions with Firebase.

Running the App

To run the app, you need to start a local server. We've provided multiple ways to do this:

Option 1: PowerShell Server (Windows)

  1. Go to the main folder where you extracted the files
  2. Double-click on start-powershell-server.bat
  3. Open your browser and go to http://localhost:8080

Option 2: Install Node.js

  1. Download and install Node.js from nodejs.org
  2. After installation, return to the app folder
  3. Double-click on start-server.bat
  4. Open your browser and go to http://localhost:8080

Option 3: Install Python

  1. Download and install Python from python.org
  2. After installation, return to the app folder
  3. Double-click on start-python-server.bat
  4. Open your browser and go to http://localhost:8080

Firebase Configuration

Make sure your firebase-config.js file exists in the public folder and has the correct Firebase credentials.

For more detailed instructions, please refer to the HOW_TO_RUN.md file in the main directory.
